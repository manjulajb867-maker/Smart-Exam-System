# Smart Examination System

## 📌 Description
This project is a Python-based Smart Examination System.

## 🚀 Features
- Exam Creation
- Student Seat Allocation
- Automatic Evaluation
- Result Generation
- JSON File Storage
- Synchronization using Threading

## 🛠 Technologies Used
- Python
- JSON
- Threading

## ▶️ How to Run
1. Run the Python file
2. Create exam
3. Register student
4. Take exam
5. View results

## 🌐 Live Demo
Open in Google Colab and run step-by-step.
